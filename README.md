# React + TypeScript

## Getting Started

Example project of implementing google books api with vite, react and react-route-dom-v6.

![2023-10-25 16.04.01.gif](./preview.gif)

## How to run

1. Clone this repository
2. Run `pnpm install`
3. Run `pnpm dev`

## How to build

1. Run `pnpm build`
